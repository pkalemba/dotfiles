# Paweł's Dotfiles

## Requirements

* Debian/Macos

* Git

## What you will get

* Zsh with oh-my-zsh

* Tmux

* [Kitty](https://github.com/kovidgoyal/kitty) Terminal

* Neovim

## How To Install

* Clone this repository:

```git clone https://github.com/pkalemba/dotfiles ~/.dotfiles```

* Enter cloned repo

 ```cd ~/.dofiles```

* Start installation process

```./install```
